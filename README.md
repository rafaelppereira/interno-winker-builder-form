# Winker builder form

Essa aplicação foi criada com o objetivo de mostrar um exemplo de como funciona a biblioteca CraftJS





## Instalação

#### - Pré-requisitos
    # - Instalação do Node js

```bash
  npm install ---------- Instala as dependências
  npm run dev ---------- Executa o projeto em dev
```
